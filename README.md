# chestnut

For Windows you need to install [Microsoft Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe).

## Quick Start

Clone:

    git clone https://github.com/andreymlv/chestnut.git --recurse-submodule
    cd chestnut

Build:

    mkdir build
    cd build
    cmake ..
    make -j12
